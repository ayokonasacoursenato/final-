About

Network Management refers to the process of administering, controlling, and maintaining a computer network. The primary goal of network management is to ensure the smooth and efficient operation of a network, whether it's a local area network (LAN), wide area network (WAN), or a combination of networks.

Key aspects of network management include:


a) Configuration Management - Configuration management refers to the process of systematically handling changes to a system's configuration in a way that preserves integrity and trace ability. In the context of IT and network management, configuration management involves managing and controlling the configuration of hardware and software components within an IT infrastructure.


Examples of configuration management in IT include:


I. Version Control Systems: These systems manage changes to source code and other software artifacts. Examples include Git, Subversion (SVN), and Mercurial.

II. Configuration Management Databases (CMDB): A centralized database that stores information about configuration items and their relationships. CMDBs help organizations track and manage configuration data.

III. Automated Deployment Tools: Tools that automate the deployment of software and configurations to different environments. Examples include Ansible, Puppet, and Chef.


Key aspects of network management include:


b) Networking Monitoring - Network monitoring involves the continuous observation and analysis of a computer network's performance, health, and security to ensure its optimal functioning. The goal of network monitoring is to detect issues, identify their root causes, and take corrective actions to maintain the network's reliability and performance.


Examples of network monitoring tools include:

I. Wireshark: A widely-used open-source packet analyzer that allows network administrators to capture and analyze the data traveling back and forth on a network.

II. SolarWinds Network Performance Monitor (NPM): A commercial network monitoring tool that provides real-time visibility into network performance, detects issues, and offers performance analysis.

III. Cisco Prime Infrastructure: A network management tool specifically designed for Cisco networks, offering features such as device management, monitoring, and troubleshooting.
